# Process Reference Targets

The process benchmark family uses Tennessee Eastman style public references plus
repository-curated supervisory command cases for the PLC-backed asset family.

## Acquisition note

This path is reserved for frozen reference material, notes, and any lightweight
artifacts used to justify process operating envelopes, fault motifs, and
interlock rationale.

The benchmark release should not present this source as a raw TE dataset
evaluation. It is a provenance anchor for supervisory process-interlock cases.
